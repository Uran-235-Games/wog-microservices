# Сервис состоит из 2 частей: Battle_service и Match_making

### Battle_service:
* сохраняет и хранит обьекты игр и запросов, mongoDB
* генерирует id для каждой игры и запроса
* предоставляет доступ к обьекту игры либо запроса по id

### Match_making
* Сохраняет прямые офлайн запросы, mysql
* Выдает список прямых запросов по id юзера
* В будущем: подберает соответствующих соперников
